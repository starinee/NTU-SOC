# Strict Release Contents

This release tree keeps the strict matched temperature pipeline as the only
official result source for the thesis.

Included:

- strict matched temperature pipeline code
- the shared within-temperature base script required by the strict pipeline
- final paper table generation code
- deployment-oriented proxy validation code
- official strict summary CSV/MD files
- pooled metrics CSV/MD files for reproducibility checks
- strict run metadata
- final 25degC paper tables
- deployment-oriented proxy outputs
- strict transfer figure and selected-row manifests
- strict 25degC checkpoints needed by the deployment proxy scripts

Excluded:

- legacy 25degC result folders
- older non-strict temperature-transfer outputs
- large per-profile prediction CSV dumps that are regenerated by the pipeline
- raw MAT datasets and local virtual environments

The paper tables use `strict_matched_test_average.csv`, which reports the
unweighted mean of profile-wise test metrics over UDDS, LA92, and NN. Metrics
computed over concatenated test samples are stored separately in
`strict_matched_pooled_metrics.csv`.
