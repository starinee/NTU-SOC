# Temperature Transfer Outputs

This folder contains the full temperature-transfer experiment generated by:

`./run_temperature_transfer_experiments.py`

Main outputs:

- `temperature_transfer_test_average.csv` and `.md`: average RMSE/MAE over UDDS, LA92, and NN for each train-test temperature pair.
- `temperature_transfer_profilewise_metrics.csv` and `.md`: profile-wise metrics for each individual drive profile.
- `temperature_transfer_full_summary.csv`: full test and average summary.
- `figures/temperature_transfer_rmse_grouped.png`: grouped RMSE comparison across train-test temperature pairs.
- `figures/temperature_transfer_rmse_heatmap.png`: heatmap view of RMSE degradation under temperature transfer.
- `selected_rows/`: exact train/test files used for 25degC, 10degC, and 0degC.
- `train_25degC/`: 25degC-trained model checkpoints, training histories, and prediction files for 25degC, 10degC, and 0degC tests.

Formal transfer setting:

- Training at 25degC uses Cycle_1-Cycle_4 and US06.
- Testing uses UDDS, LA92, and NN.
- Cross-temperature transfer evaluates 25degC-trained models on 10degC and 0degC constant-temperature tests.
- Within-temperature references for 10degC->10degC and 0degC->0degC are imported from `./full_temperature_rerun_outputs`.

Legacy single-model files named `temperature_transfer_mlp_filtered_*` are from an earlier filtered-feature MLP-only check and should not be used as the formal full-transfer result.
